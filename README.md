# GenAI LangChain

A hands-on repository to learn and build Generative AI applications using LangChain and Large Language Models (LLMs). This repository contains beginner-friendly notebooks and examples covering core LangChain concepts, Retrieval-Augmented Generation (RAG), AI agents, retrievers, vector stores, and more.

## 🚀 What You'll Learn

- Understanding LangChain fundamentals
- Chaining LLM calls using LangChain
- Working with Prompt Templates
- Output Parsers and Structured Outputs
- Building AI Chatbots and Assistants
- Retrieval-Augmented Generation (RAG)
- Vector Databases and Retrievers
- Tool Calling and AI Agents
- Integrating APIs and External Data

---

## 📂 Repository Structure

### 📁 langchain-chains-main
Learn different types of LangChain chains and chaining techniques.

### 📁 langchain-document-loaders-main
Load PDFs, text files, websites, and external documents into LangChain.

### 📁 langchain-models-master
Working with LLMs and chat models inside LangChain.

### 📁 langchain-output-parsers-main
Parse and structure LLM outputs efficiently.

### 📁 langchain-prompts-main
Prompt engineering and prompt templates using LangChain.

### 📁 langchain-runnables-main
Understanding runnables and execution pipelines.

### 📁 langchain-structured-output-main
Generate structured outputs from LLMs.

### 📁 langchain-text-splitters-main
Different text splitting strategies for chunking documents.

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/ProxyCode1010/GenAI-LangChain.git
```

Move into the project folder:

```bash
cd GenAI-LangChain
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
